Intro to IRC

Registering allows you to create channels and protects your nickname from unauthorized use.

Register a nick

/msg NickServ register your_password email@address.com

Identify yourself

/msg NickServ identify your_password

NickServ help

/msg NickServ help

Create a channel

If you need a channel simply use the commands below with a properly identified nickname.

/msg ChanServ register #channel_name

ChanServ help

/msg ChanServ help
